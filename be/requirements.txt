# ========================
# NN Auth System — Dependencias del Backend
# ========================
# ¿Qué? Lista de todas las dependencias Python del backend con versiones exactas.
# ¿Para qué? Garantizar instalaciones 100% reproducibles y sin CVEs activos.
# ¿Impacto? Versiones sin pinear (>=, ~=, *, ^) permiten que pip instale automáticamente
#           versiones futuras con regresiones o vulnerabilidades. Versiones exactas (==)
#           aseguran que todos los entornos ejecuten EXACTAMENTE el mismo código auditado.
#
# REGLA DE ORO — PROHIBIDO en este archivo:
#   ✗  fastapi>=0.115.0   (permite cualquier versión futura, incluso con CVEs)
#   ✗  fastapi~=0.115.0   (permite actualizaciones de parche no auditadas)
#   ✗  fastapi            (sin versión = ruleta rusa)
#   ✓  fastapi==0.135.1   (única versión auditada y aprobada)
#
# Para actualizar una dependencia:
#   1. Revisar changelog y CVEs de la nueva versión
#   2. Actualizar la versión aquí
#   3. Ejecutar `pip install -r requirements.txt` y `.venv/bin/pip-audit`
#   4. Ejecutar `pytest -v` para confirmar que los tests siguen pasando
#   5. Commit con mensaje: "chore(deps): upgrade <paquete> X.Y.Z → A.B.C"
